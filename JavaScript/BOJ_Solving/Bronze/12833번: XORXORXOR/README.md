## 문제

[문제 링크](https://boj.kr/12833)

<p>세 수 A, B, C를 입력 받은 다음, ( ( ( ( A XOR B ) XOR B ) XOR B ) … ) XOR B 형태로 연산을 C회 했을 때의 결과값을 출력하는 프로그램을 작성하시오.</p>

## 입력

<p>첫째 줄에 A, B, C가 주어진다. (0 &lt; A, B, C ≤ 10<sup>9</sup>)</p>

## 출력

<p>첫째 줄에 계산된 결과를 출력한다.</p>
